# FurgoRent: Instrucciones del Proyecto

## Stack Tecnológico
- Framework: Next.js 15 (App Router)
- Base de Datos/Auth: Supabase
- UI: Tailwind CSS + Shadcn/UI
- Pagos: Stripe Connect

## Roles de Agente
- **Arquitecto:** Responsable de la estructura de datos y seguridad (RLS).
- **UI/UX:** Responsable de componentes visuales profesionales y responsive.
- **Integrador:** Responsable de conectar APIs (Stripe, Maps, Supabase).

## Reglas de Oro
1. Usar TypeScript estricto en cada archivo.
2. Antes de crear una tabla nueva, verificar el esquema actual.
3. Los componentes de UI deben seguir el estilo de Shadcn/UI.
